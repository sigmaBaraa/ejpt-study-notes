# Content Review

## Review Result

The original Notion export was inspected before creating this repository. It contains an `eJPTv2 Mastery Hub` with Markdown files, CSV metadata, and one image asset.

The export includes material that appears unsafe to publish directly:

- official certification/course descriptions
- course-note style sections marked with INE fields
- links and references to external repositories and third-party hosted images
- large command-heavy notes that may be derived from training content

The imported notes were restored into `notes/source-notes/` after the user confirmed that external sources are acceptable. The files were renamed from Notion-generated names into readable slugs.

## Publishing Decision

This repository includes the restored source notes plus a safer project structure around them. For long-term public use, keep improving the imported files into original work:

- original summaries written in your own words
- personal lab reflections
- methodology checklists created from your own practice
- command references that you personally validate and explain
- diagrams or screenshots that you created yourself

## What To Review Before Public Promotion

Review and rewrite any copied course paragraphs, paid training text, or third-party screenshots unless you have explicit permission.

## Recommended Rewrite Approach

For each original note:

1. Keep the topic title.
2. Remove copied course wording.
3. Rewrite the concept in your own words.
4. Add a short "When I used this" lab note.
5. Include only commands you understand and can explain.
6. Link to official documentation where possible.
