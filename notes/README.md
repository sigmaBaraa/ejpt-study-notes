## Recommended Note Format

Use this structure for each future note:

```markdown
# Topic Name

## Goal

What this topic helps you accomplish.

## Key Concepts

Short original explanation.

## Practical Workflow

Validated steps from your own lab practice.

## Commands

Commands you tested and understand.

## Defensive Takeaway

What a defender should monitor or harden.

## References

Official docs or public references.
```
